# Vantage Point AI | Cricket DRS System

Vantage Point AI is a high-fidelity Cricket Decision Review System (DRS) powered by Google Gemini 2.5 Flash Lite and Firebase. It provides elite-level visual analysis for cricket officiating, including wicket checks, catch validity, and crease integrity.

## Key Features

- **AI Analysis Terminal**: Real-time visual processing of delivery footage using Gemini 2.5 Flash Lite.
- **Precision Verdicts**: "LOUD and DECISIVE" umpire assistance powered by multimodal AI.
- **Live Match Monitor**: A premium, IPL-style broadcast scoreboard featuring real-time T20 data via CricAPI.
- **Cinematic UI**: High-impact visuals with Barlow typography and neon-glow interfaces.

## Technology Stack

- **Framework**: Next.js 15 (App Router)
- **AI Engine**: Google Genkit + Gemini 2.5 Flash Lite
- **UI Components**: Radix UI + ShadCN
- **Styling**: Tailwind CSS + Custom Neon Glow Effects
- **Fonts**: Barlow & Barlow Condensed (Sports Broadcast standard)
