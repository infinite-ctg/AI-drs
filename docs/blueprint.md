# **App Name**: Vantage Point AI

## Core Features:

- Intelligent Review Engine: Upload delivery videos or screenshots to be processed by a custom vision tool using Gemini API and OpenCV to detect ball trajectory and crease boundaries.
- Broadcast Verdict Graphics: Dynamic, cinematic reveal screens featuring 'OUT' or 'NOT OUT' verdicts with stadium-light glowing effects and high-fidelity transitions.
- Frame-by-Frame Scrubbing: A professional-grade replay timeline that allows users to analyze AI-highlighted frames such as foot-on-line or ball-on-glove contact.
- Review Decision Vault: Secure storage of past decision reviews and analysis metadata using Firestore for historical comparisons and social sharing.
- Scriptwriter’s Narrative AI: A parody mode feature that generates fictional, dramatic commentary scripts using match context tools for satirical entertainment.
- Essential Match Monitor: A compact, high-contrast scoreboard widget providing real-time secondary context without cluttering the main review experience.
- AI Logic & Confidence Readout: A detailed sidebar displaying 'AI-assisted prediction' confidence scores and reasoning summaries to provide transparent logic behind verdicts.

## Style Guidelines:

- Primary color: Electric Indigo (#6366F1) inspired by digital night-time broadcasts and high-energy stadium LEDs.
- Background color: Deep Midnight (#0F1117), a desaturated indigo providing a rich cinematic foundation for neon UI elements.
- Accent color: Vibrant Orchid (#A855F7), used for key highlights and secondary neon accents, ensuring visual contrast and high energy.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and scoreboard metrics to evoke a tech-driven feeling; 'Inter' (sans-serif) for technical analysis and summaries.
- Sleek, monolinear icons with subtle inner glows to match the glassmorphism aesthetic of premium OTT sports packages.
- An immersive wide-screen dashboard layout featuring a central cinematic media player surrounded by technical readouts and modular telemetry widgets.
- Fluid framer-motion transitions with dramatic slow-motion expansions and high-tension 'CHECKING...' loading states before verdict reveals.
